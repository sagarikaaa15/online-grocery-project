# Online Grocery Project

A full-stack online grocery shopping web application that allows users to browse products, search items, add products to the cart, and manage orders.

## Features
- Product listing
- Search functionality
- Category filtering
- Shopping cart
- Order management
- Responsive UI

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express.js

## How to Run

1. Clone the repository:
```bash
git clone https://github.com/sagarikaaa15/online-grocery-project.git
```

2. Install dependencies:
```bash
npm install
```

3. Start the server:
```bash
npm start
```
